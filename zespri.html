This webapp is a data management tool. The tool is hooked up to a database. The idea is that you can select a topic from the menu (which flips open) and the requested graph displays, giving a nice visual tool for following the business.

I used vanilla JS and GraphsJS. GraphsJS is a JS library. It provides a great solution for turning dry numbers into something more useful.

Deze webapp is een data management tool. De tool wordt gekoppeld aan een database. Als je een selectie maakt uit het uitklapbare menu krijg je de data in grafiekvorm te zien. Deze visuele feedback is van onschatbare waarde voor de business.

Ik heb hiervoor Vanilla JavaScript en GraphJS gebruikt.
