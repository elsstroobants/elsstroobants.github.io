Veggiezz Veggiezz

This is my modest contribution towards a world where more people become aware of the climate challenges whilst still being on friendly terms with their taste-buds.

React JS was used to build the app. The intention is to couple it up to a recipe API, and only vegetarian recipes will be returned in the search results.

Dit is mijn bescheiden bijdrage aan een wereld waarin minder vlees en meer groente gegeten zou moeten worden. Je zoekt op basis van een groente en krijgt een lijst met opties waar je smaakpapillen blij van worden. Vervolgens kan je de recepten bekijken en toevoegen aan je favorieten lijst.

Voor het ontwikkelen van deze app heb ik React JS gebruikt. Door de app te koppelen aan een recepten API krijg je tonnen inspiratie op het vlak van vegetarisch koken.
