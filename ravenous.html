Ravenous Ravenous

This app helps you to find restaurants based on their location, category and customer rating.

This app was made by using React JS. React JS is a JavaScript library that makes it possible to work with reusable components. Very simple, efficient and fast, and if you ask me, definitely the way to go for this type of app.

Met deze app kan je zoeken naar restaurants op basis van hun ligging, categorie en bezoekersscore.

Deze app is gemaakt met gebruik van React JS. React JS is een JavaScript library die het mogelijk maakt om met herbruikbare componenten te werken. Heel eenvoudig en snel en daarom, als je het mij vraagt, de 'way to go' voor dit soort apps.
